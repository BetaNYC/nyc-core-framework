Make beautiful, accessible websites.

The NYC Core Framework is an open-source front end toolkit for building modern websites and browser-based applications. Designed to remove technical barriers for teams with limited resources, we provide a wealth of prebuilt front-end components customizable for projects of any size and complexity.

The Framework at a Glance

The NYC Core Framework is built with Sass, CSS Custom Properties, and modern Javascript. The Framework proper is made up of the following folder and file structure.

CSS and JS Dependencies

In Figure 1 (above), notice the highlighted files: /dist/css/nyc_core.css (line 4), /dist/js/nyc_core.js (line 6).

These CSS and JavaScript files are the only two dependencies generated from the NYC Core Framework, and they are available to be used via CDN (Content Delivery Network).

You don't need to set up a complicated development environment to use the NYC Core Framework. Follow the Quick Start CDN intructions below to get started with our CDN.
