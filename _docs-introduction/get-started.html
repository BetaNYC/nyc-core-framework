Make beautiful, accessible websites.

The NYC Core Framework is an open-source design system for building modern websites. The Framework aims to remove technical barriers for those with limited resources. It provides a wealth of customizable components accessible via CDN.

At the same time, the framework is ideal for seasoned developers and complex projects. Install it with NPM, customize your build with Webpack, or use it in your next a React App.

Framework at a Glance

We build and maintain the Framework using modular CSS (SASS) and JavaScript (es6+). Figure 1 (bellow) depicts the folder and file tree that make up the Framework proper:

Each module is a folder within the /src/modules/ directory. We use Webpack to bundle all modules into one CSS file, and one JavaScript file. This helps maintain (and future-proof) the framework while accounting for out-of-date browsers.

CSS and JS Dependencies

In Figure 1 (above), notice the highlighted files: /dist/css/nyc_core.css (line 4), /dist/js/nyc_core.js (line 6).

These files are two of the three required dependencies of the NYC Core Framework. The third dependency is for icons. Follow our Quick Start instructions (below) to learn how to use each of the three dependencies.
