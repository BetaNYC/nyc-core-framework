Make beautiful, accessible websites easier and faster.

The NYC Core Framework is an open-source front end toolkit for building modern websites and browser-based applications. Our lightweight codebase is easy to understand—and at the same time—provides a wealth of accessible components ideal for projects of any size and complexity.

Framework at a Glance

The NYC Core Framework includes the following files and folders.

CSS and Javascript

In Figure 1 (above), pay attention to the highlighted files: /dist/css/nyc_core.css (line 4), /dist/js/nyc_core.js (line 40).

These CSS and Javascript files are the only two dependencies generated from the NYC Core Framework, and they are available via CDN (Content Delivery Network). Follow our Quick Start Guide (in the next section) to get started with these core files.

Directory Structure

At the root of the Framework, the /css and /js directories each contain the same three subdirectories: /dist, /lib, and /src.
