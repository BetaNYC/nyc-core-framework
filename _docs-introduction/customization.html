Distinguish yourself without starting from scratch.

There are two different ways to customize the NYC Core Framework—with our CDN stylesheet and CSS custom properties, or with a Sass development environment.

CDN and Custom Properties

Our Quick Start Guide shows you how to include the Framework's core stylesheet via CDN. Within he Framework, a handful of our styles are declared using custom properties (also known as CSS variables).

If you're unfamiliar with custom properties, read the MDN web docs. They are similar to Sass variables because they both allow you to store and reuse values in your CSS (see Figure 6 and 7 below). Custom properties, however, provide additional benefits that Sass variables do not.

Because custom properties update at runtime, we can modify values from our CDN stylesheet by declaring new values in a second stylesheet. In the next exercise, you'll learn how to customize your theme colors using this methodology.
