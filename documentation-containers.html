Narrow, Medium, and Wide

Designed for use alongside Boostrap's native containers, rows, and columns, .narrow, .medium, and .wide classes can be used in a number of ways for enhanced control of your content.

Narrow

Narrow uses a CSS character width unit (ch) of 55, ideal for maximum readability of paragraph text. Place .narrow on block-level elements, like divs and sections — not directly to text elements like headers and paragraph tags. This ensures narrow's width remains consistent, calculating it's width from a single, base font-size.

Medium

Medium's width is a multiple (1.25x) of narrow's width. Use .medium for figures or any content you wish to appear slightly wider than narrow.

Wide

Wide can be used on .container-fluid or any element where you wish to constrain the width. Wide's with is set to 2400px.

Variables for narrow, medium, and wide are set in scss/_a-global-variables.scss under D. Supplementary widths, while their classes are declared in scss/_e-supplementary-classes.scss under E. Supplementary widths.

Matrix Rows

Extend Bootstrap rows with .matrix-border and .matrix-gutter classes. These classes provide consistent spacing around all sides of a row's columns, regardless of the column's individual width.

These classes allow to quickly create dynamic, responsive grids. Further, you won't need to set the top or bottom margins when columns are stacked (on mobile view, for example).
