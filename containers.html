Narrow, Medium, and Wide

Designed for use alongside Boostrap's native containers, rows, and columns, .narrow, .medium, and .wide classes can be used in a number of ways for enhanced control of your content.

Narrow

Narrow uses a CSS character width unit (ch) of 55, ideal for maximum readability. For best results, place .narrow on block-level elements, like divs and sections — not directly to text elements like headers and paragraph tags. In this way, narrow's width is calculated from our base font-size.

Medium

Medium's width is a multiple (x 1.25) of narrow's width. Use .medium for figures or any content you wish to appear slightly wider than narrow.

Wide

Wide can be used on .container-fluid or any element where you wish to constrain the width. Wide's with is set to 2400px.

Variables for narrow, medium, and wide are found in scss/_a-global-variables.scss under D. Supplemental widths, while these classes are found in scss/_e-supplementary-classes.scss under E. Supplementary widths.

Matrix Rows

Extend Bootstrap rows with .matrix-border and .matrix-gutter classes. These classes provide consistent spacing around all sides of a row's columns, regardless of the column's individual width.

These matrix classes allow to quickly create dynamic grids. Further, won't need to set the top or bottom margins when columns are stacked (on mobile view, for example).
