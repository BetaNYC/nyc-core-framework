Narrow, Medium, and Wide

Designed for use alongside Boostrap's native containers, rows, and columns, .narrow, .medium, and .wide classes can be used in a number of ways for enhanced control of your content.

Narrow

Medium

Wide

Narrow

Narrow uses a CSS character width unit (ch) of 55, ideal for maximum readability. For best results, place .narrow on block-level elements, like divs and sections — not directly to text elements like headers and paragraph tags. In this way, narrow's width is calculated from our base font-size.

Medium

Medium's width is a multiple (x 1.25) of narrow's width. Use .medium for figures or any content you wish to appear slightly wider than narrow.
