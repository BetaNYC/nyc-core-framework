Aspect Ratios

Our responsive aspect-ratio boxes are perfect for embedding video, interactive maps, and anything else where a width to height ratio should be maintained.

Available Sizes

Aspect Ratios come in the sizes shown in the table below.

Each size has a unique class name: .aspect-ratio--{w}by{h}, where w is the width and h is the height. These classes are generated by the $aspect-ratios Sass map located in /scss/base/_aspect-ratios.scss. under Section A. Variables.

Sizes in the map are represented by two parenthetical numbers where the first value represents the width and the second value represents height. You can edit or remove these sizes, or add new sizes to the map.

Example HTML

Aspect ratio boxes have a parent and child selector. On the parent, apply the class with the size modifier .aspect-ratio--{w}by{h}. The child selector requires the class .aspect-ratio__content.
