Supplementary widths

Narrow, Medium, and Wide

Designed for use alongside Boostrap's native containers, rows, and columns, .narrow, .medium, and .wide supplementary classes can be used in a number of ways for enhanced control of your content.

Narrow

Medium

Wide

Narrow

Narrow uses a CSS character width unit (ch) of 55, ideal for maximum readability of paragraph text. Place .narrow on block-level elements, like divs and sections — not directly to text elements like headers and paragraph tags. This ensures narrow's width remains consistent, because it's width is calculated from a single, base font-size (16px).

Medium

Medium's width is a multiple (1.25x) of narrow's width. Use .medium for figures or any content you wish to appear slightly wider than narrow.

Wide

Wide can be used on .container-fluid or any element where you wish to constrain the width. Wide's with is set to 2400px.

Variables for narrow, medium, and wide can be customized in scss/_a-global-variables.scss under D. Supplementary widths, while their classes are declared in scss/_e-supplementary-classes.scss under E. Supplementary widths.
