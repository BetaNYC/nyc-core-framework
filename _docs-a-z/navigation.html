Primary Nav Component

Designed for use within the document's primary header tag, our primary nav component is a complete menu system made up of three preassigned areas: the logo (wrapped within a home link), global action buttons, and of course, the menu itself.

The primary nav's parent selector requires two chained modifiers (.primary-nav--{x}--{y}) where x provides the menu placement (as it relates to the logo inline or below), and y provides the responsive breakpoint (e.g., lg) at which the component changes layout from mobile to desktop.

With the example above, the parent selector class is .primary-nav--below--lg.

Basic Navigation

Add the class .nav to a <ul> for a simple vertical menu.

For a horizontal menu, apply the .nav--horizontal modifier class.

The .nav--horizontal modifier class accepts an additional (chained) breakpoint modifier (e.g., --md). With the .nav--horizontal--md class present, the navigation below starts as a vertical menu and switches to a horizontal menu at the medium breakpoint.

Justification

Use our flex classes for centered or right-aligned navigation.

Change the link color, font size, and font-weight by adding utility classes directly to any ancestor.

Bordered Navigation

Add the .nav--divider modifier class to a .nav selector for vertical, bordered navigation menu.

The horizontal and breakpoint modifier classes (.nav--horizontal and .nav--horizontal--{bp} where bp is the breakpoint shorthand) also apply to .nav-divider selectors.

Add an outer border to the menu with our border classes.
