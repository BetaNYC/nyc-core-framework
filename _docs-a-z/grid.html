A traditional 12 column flexbox grid that works consistently across browsers, including Internet Explorer 11.

Grid Selector Classes

Grids require a parent selector with a class of .grid, and a child (or children) with a class of .grid__cell.

Column Width

Grid columns occupy 100% of the available width (stacked one above another) until otherwise specified. For equal-width columns, add the .grid--cells modifier to the .grid selector. This class distributes its child columns evenly based on the number of columns present. With three columns present in the example below, columns are divided into thirds.
