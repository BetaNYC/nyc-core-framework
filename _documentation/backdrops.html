--- layout: documentation title: "Backdrops" ---

A panorama of possibilities.

Natura11y's Backdrop components place a translucent image behind any custom content. Backdrops offer two distinct variations: an image-proportional backdrop that scales and stacks responsively, and a fixed-height backdrop that can adjust to the height of its inner content.

Image Accessibility

Backdrops replace CSS background images with inline HTML images so screen readers can read the image's alt text.

Learn more about accessible images.
