Make beautiful, accessible websites easier and faster.

The NYC Core Framework is an open-source front end toolkit for building modern websites and browser-based applications. Our lightweight codebase is easy to understand—and at the same time—provides a wealth of accessible components ideal for projects of any size and complexity.

CSS Custom Properties

In NYC Core Framework, colors, fonts, and other styles can be customized using CSS custom properties (CSS Variables).

It's no longer required to set up a Sass development environment to apply your own styles to the Framework. In the Quick Start Guide (above) we include the NYC Core stylesheet via CDN.

You have the ability to update certain properties defined in this stylesheet by redefining them in a second stylesheet. To see how this works, create a new CSS document and name it theme-properties.css, then save it to the /css/dist/ directory.

Next, open your HTML document and add a link to theme-properties.css below nyc_core.css.
