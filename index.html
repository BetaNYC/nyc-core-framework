Getting Started

The NYC Core framework requires modern Bootstrap (4.x) for styling, Font Awesome for icons, and custom CSS (SASS) and Javascript for accessibility. Google Translate is included for language translation, and right-to-left languages are supported out-of-the-box.

All of these dependencies are included in the starter files below.
