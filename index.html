Getting Started

The NYC Core framework combines Bootstrap (4.x) for styling, Font Awesome for icons, Google Translate for languages, and our own SCSS and Javascript for out-of-the-box accessibility.

Everything you need to get started using the NYC Core Framework is included in the starter files. Check the docs for features and code examples.

Download Starter Files

Starter Files at a Glance

  • bootstrap
    • scss Bootstrap's (unedited) SCSS dependencies.
  • css
    • theme.css Your theme's compiled stylesheet.
  • js
    • main.js Your theme's main js file. Required for right-to-left language translation and link translation in the global footer.
  • scss
    • __header.scss Adds legal information to theme.css and must be included for redistribution.
    • _a-global-variables.scss Your theme's global variables (required before Bootstrap imports).
    • _b-bootstrap-imports.scss Imports Bootstrap partials (only what we need).
    • _c-custom-mixins.scss A handful of SASS mixins not provided by Bootstrap.
    • _d-enhancements.scss Accessibility enhancements, margin resets, cross-browser bug fixes, and more.
    • _e-supplementary-classes.scss Handy classes not provided by Bootstrap.
    • _f-ui-elements.scss Custom user-interface elements either not provided by Bootstrap or extend Bootstrap's component library.
    • _g-dev-tools.scss A few nice-to-haves for developers.
    • _h-nyc-subway-icons.scss NYC subway icons.
    • theme.scss Imports all SASS dependencies. The best place to add your own styles.
  • example-home-nav-across.html NYC header and footer with main navigation across (includes all dependencies).
  • home-nav-right.html NYC Header and Footer with main navigation right (includes all dependencies).
  • images
    • nyc-logo-footer.svg Official NYC logo for global footer.
    • nyc-logo-footer.svg Official NYC logo for global header (top-right).
